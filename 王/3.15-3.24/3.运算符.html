赋值运算符：= += -= *= %= /=

例如：c+=1:c=c+1;c-=1:c=c-1等

算术运算符：+ - * / % 但+运算符若两边不都是数字，则为拼接符号

如‘我的名字叫’+userName

关系运算符：< > <= >= ==(相等：值是否相等，具有隐式转换)

===（全等：值和类型相等，不具有隐式转换)

！=（不相等） ！==（不全等） 输出的结果为布尔值 true or flase

特殊的比较:console.log(null==undifined);console.log(true==1);console.log(flase==0);

逻辑运算符 &&与 |或（有一个满足即可） ！非（无论是什么类型，都会返回布尔值）

布尔口诀：数字非0即真 字符串非空即真 undifined null 的布尔值都为flase

一元运算符：++ a++表示先赋值再自加 ++a表示先自加再赋值

a=1;

b=a++ 则b=1，a=2

c=++a 则c=3，a=3
